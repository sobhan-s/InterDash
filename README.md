# InternDash

A React + TypeScript dashboard application built with Vite, Tailwind CSS, and shadcn/ui. Needs a LOT of love.

**Live:** https://interdash.vercel.app/

## Setup

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Run development server
- `npm run build` - Production build  
- `npm run analyze` - Bundle analysis (`npx vite-bundle-visualizer`)
- `npm run preview` - Preview production build
- `npm run lint` - ESLint

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui components
- Chart.js + Recharts
- Three.js + React Three Fiber
- D3.js
- mathjs
- moment + lodash (full imports)
- pdf-lib, xlsx, papaparse, jszip
- @faker-js/faker
- crypto-js
- highlight.js + react-syntax-highlighter

## Features

- Crypto price tracker (CoinGecko API)
- Weather dashboard (Open-Meteo API)
- User management (JSONPlaceholder)
- Posts feed with comments
- Todo list
- Charts & analytics (Chart.js + Recharts)
- Photo gallery (5000 images)
- Markdown editor with live preview
- 3D visualization (Three.js)
- D3 force graph
- Math playground
- Report generator (PDF, Excel, CSV, ZIP)
- Search & filter
